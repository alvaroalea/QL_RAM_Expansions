##Instructions

### COMPONENTE REQUIERED

8 or 16  41256 RAM chips 120 to 150nS SPEED.

If fitting 8 RAMS (256K) the row closest to the edge connector should be used.

Pin One of the RAM chips is furthest from above edge connector.
